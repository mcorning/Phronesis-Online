
# Updated Wildfire Mitigation Roadmap (Including Innovations)

## Immediate Actions
- Install deck-to-wall flashing (2" rise, 1" overhang)
- Enclose deck undersides with 1/8" metal mesh
- Create noncombustible 0–5 ft zone
- **Install Ember Grate (Innovation #1)**
- **Pre‑evacuation Sprinkler Plan (Innovation #2)**
- **Open Deck Gates (Innovation #3)**

## Near-Term Improvements
- Maintain 6" siding clearance
- Remove debris from roof and decks
- Clean 0–5 ft zone weekly

## Mid-Range Actions
- Tree pruning and ladder fuel reduction
- Outbuilding buffers (5 ft NCZ)

## Evacuation Kit
- Sprinklers staged and hoses attached
- Gates labeled for quick opening
